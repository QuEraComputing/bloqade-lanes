Gate region spacing pattern

Memory region spacing pattern
Interleaved x: 0, 4, 8, 12, 16, 20, 24, 28
Gaps: 4, 4, 4, 4, 4, 4, 4
Uniform spacing — no tight pairs
Word slicing — global template (same in every zone)
Word 0: [(0,0), (1,0)]  top row, left pair    Word 1: [(2,0), (3,0)]  top row, right pair
Word 2: [(0,1), (1,1)]  bottom row, left pair    Word 3: [(2,1), (3,1)]  bottom row, right pair
4 words × 2 sites = 8 sites per zone, horizontal pairs tiling a 4×2 grid
Key insight
The spacing pattern reveals the physics: in the gate region, alternating tight/wide gaps show where entangling pairs sit (2.0 apart). In the memory region, uniform gaps show storage without entangling capability. Same logical structure, different physical geometry — this is why zones must own their own Grid.
